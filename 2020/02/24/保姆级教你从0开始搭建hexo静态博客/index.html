保姆级教你从0开始搭建hexo静态博客

Posted on 2020-02-24

date: 2020-02-24 15:17:27
前言：新手小白的第一个blog，先简单搭建起来，之后大家也可以自己考虑美观，页面内容等问题。各位自行搭建过程中可能会出现一些问题，但是不必畏惧，网上关于hexo有很多健全的搭建教程和问题解决方案。大家勇于百度勇于重来，加油，奥利给。

一、搭建环境

1、安装NodeJs

  下载地址：https://nodejs.org/en/ (可下载LTS长期支持版)
  安装过程一直next就好啦
  win+r 键输入cmd调出命令提示符,输入node -v 能查看到版本号即安装成功

2、安装Git

  官网下载地址：https://git-scm.com/
  tips:官网下载速度很慢可以在淘宝的一个镜像的网站中下载
       https://npm.taobao.org/mirrors/git-for-windows/
  下载完打开输入git version 查看是否安装成功

二、安装hexo

  由于国内镜像源安装速度慢,借助npm装一个淘宝的源,输入npm install -g cnpm –registry=https://registry.npm.taobao.org 等待安装
  输入cnpm install -g hexo-cli 安装hexo
  输入hexo -v 查看版本号验证是否安装成功,之后就可以开始正式搭建博客啦

三、搭建博客

1、新建博客存储文件夹

  建立一个文件夹 mkdir blog ,且初始化博客输入hexo init 等待一段时间完成后就可以在自己创建的位置看到该文件

2、启动博客

  输入hexo s
即可在浏览器中访问默认链接http://localhost:4000, 但由于我的端口被占用，所以启动时设置其他端口

浏览器中启动效果图

3、新建文章

  输入执行 hexo n “菜鸟博客”

进入该文章目录 cd source/_posts/

四、编写博客

1、输入vim “菜鸟博客”

  编写文章的技巧网上有很多，完成之后esc+:wq 保存退出

2、编译器编译

  个人觉得vim上手有点难受:) 一键右击用sublime打开编辑

五、浏览器中查看文章

  回退到blog目录 cd ../..
  hexo clean 清除缓存文件和已生成的静态文件
  hexo g 生成静态网页
  hexo s 再次启动

  在浏览器刷新就可以看到自己写的文章辽

六、将博客部署到GitHub上

1、在github上新建一个仓库（仓库名格式为 github名.github.io）

2、打开git的命令行,在blog目录下装一个git部署插件执行 cnpm install –save hexo-deployer-git

3、输入命令行 vim _config.yml

  在文件的最下面输入如下：

4、部署到远端 hexo d

  注意： 此步骤需要输入git的账号和密码，需要设置git的global。完成之后刷新github,可以看到自己博生成的内容(如下图)

5、在浏览器地址栏输入你的仓库名查看是否成功

五、更换主题

1、下载主题

  输入执行 git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia 将主题克隆到themes文件下面(大家也可以下载其他主题)

2、输入执行 vim _config.yml修改此文件

  找到 theme:landscape将主题修改为yilia

3、重新生成

  hexo clean
  hexo g
  hexo s
  刷新本地的localhost:4000,就可以访问啦

4、主题发布到github中

  执行输入 hexo d 在github页面上多刷新几次就完成辽！
